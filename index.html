LES NFTS

Expliqués par le fondateur d'Ethereum

Dans cette vidéo, Vitalik Buterin, fondateur d'Ethereum explique brièvement à Ashton Kutcher ce qu'est un NFT.

"un NFT est un actif digital dont une seule copie existe sur la blockchain (Ethereum). Les NFTS peuvent être utilisés pour représenter de l'art, des tickets d'entrée à une conférence, ou un nom ENS qui est un comme un nom digital sur la blockchain que vous pouvez posséder. Visitez Ens.domains!

Nom de domaine pour site IPFS

Les ENS sont aussi votre nom de Domaine pour sites hébergés sur l'Internet décentralisé (IPFS) grâce à Fleek.co et Eth.limo. Voici un tutoriel en anglais (Twitter @DylanMeador).

Votre adresse Crypto

ENS vous permet de remplacer votre adresse de portefeuille crypto 0xgf454gsg61464153158gfd55... par jean.compte.eth . Commencez par installer Metamask. Ensuite connectez votre portefeuille sur Le site ENS. Une fois le nom enregistré choisissez votre "primary ENS name" dans "mon compte". Vous serez alors identifié sur les sites web3 automatiquement. ENS remplace la connexion avec email et mot de passe.

Acheter un ENS

Il est fort probable que votre nom soit déjà enregistré. Si vous le voulez absolument vous pouvez émettre une offre sur Gem.xyz. Dans "recherche" entrez ENS. Entrez ensuite le nom désiré
Si vous êtes patient, il sera bientôt possible d'acheter des sous domaines (michel.compte.eth), beaucoup moins cher que les domaines parents (compte.eth).
